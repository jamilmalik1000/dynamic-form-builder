# Smart Form Builder

A **dynamic, client-side form builder** built with **HTML, CSS, Bootstrap, and Vanilla JavaScript**.  
Allows users to create custom forms, preview them live, validate inputs, and save submissions in the browser.

## Features

- Add dynamic form fields:
  - Text, Email, Number  
  - Required toggle
- Live form preview while building
- Client-side validation with real-time feedback
- Store submissions in `localStorage`
- View stored submissions in a readable format
- Modular JavaScript structure for maintainability
